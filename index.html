Calculator

This calculator project is based on the prompt from The Odin Project Web Development 101 course. It is a basic calculator that can be operated by clicking the buttons, or by typing on your keyboard. It understands relevant order of operations, allows you to clear the memory entirely (AC) or simply backspace to delete a mistyped number.

This project was a good opportunity to expore:

  • Array methods
  • Bootstrap layouts
  • Objects
  • Event Listeners
  • Function binding
